## Erste Schritte

Die HTML5 Video API bietet eine umfangreiche Schnittstelle zur programmatischen Steuerung von Videoelementen.

## Grundlegende Steuerelemente

```javascript
const video = document.querySelector("video");
video.play();
video.pause();
```

## Erweiterte Funktionen

Sie können den Wiedergabefortschritt verfolgen, die Lautstärke ändern und benutzerdefinierte Steuerelemente mit Event-Listenern implementieren.
