## Was ist HTML?

HTML (HyperText Markup Language) ist die Standard-Auszeichnungssprache zum Erstellen von Webseiten.

## Ihre erste Seite

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
```

Lernen Sie über Überschriften, Absätze, Listen und Links, um Ihre erste Website zu erstellen.
