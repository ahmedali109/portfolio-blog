## Variablen und Datentypen

```javascript

const age = 30;

```

## Funktionen

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

```javascript
const fruits = ["apple", "banana"];
const person = { name: "John", age: 30 };
```
