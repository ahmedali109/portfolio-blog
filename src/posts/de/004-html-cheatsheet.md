## Dokumentstruktur

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content here
  </body>
</html>
```

## Textformatierung

```html
<h1>Heading 1</h1>
<p>Paragraph</p>
<strong>Bold</strong>
<em>Italic</em>
```

## Links und Bilder

```html
<a href="url">Link</a> <img src="image.jpg" alt="Description" />
```
