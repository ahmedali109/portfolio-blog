## البدء

توفر HTML5 Video API واجهة غنية للتحكم في عناصر الفيديو برمجيًا.

## عناصر التحكم الأساسية

```javascript
const video = document.querySelector("video");
video.play();
video.pause();
```

## الميزات المتقدمة

يمكنك تتبع تقدم التشغيل وتغيير مستوى الصوت وتنفيذ عناصر تحكم مخصصة باستخدام مستمعي الأحداث.
