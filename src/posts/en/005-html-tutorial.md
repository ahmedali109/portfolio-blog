## What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

## Your First Page

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
```
